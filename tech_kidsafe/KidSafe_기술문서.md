## 1. 프로젝트 개요

- 프로젝트명: KidSafe (AI Media Guardian for Kids)
- 한 줄 소개: AI가 어린이 콘텐츠의 안전도를 실시간으로 검수하고, 연령과 관심사에 맞는 영상을 추천해주는 웹 플랫폼
- 개발 기간: 진행 중 (1인 개발)
- 목적: 프론트엔드 취업 포트폴리오

## 3. 핵심 기능 및 구현 기술

### 3-1. AI 콘텐츠 안전도 검수
- **사용 기술**: Anthropic API (claude-sonnet-4-5)
- **분석 항목**: 폭력성, 언어, 선정성, 교육성 (각 0~100점)
- **입력**: 영상 제목 + 설명 텍스트
- **출력**: totalScore + 항목별 점수 + AI 한 줄 요약
- **특징**: 단순 키워드 필터링이 아닌 LLM 기반 맥락 이해 분석

### 3-2. 연령별 콘텐츠 필터링
- **사용 기술**: safetyFilter.js (JavaScript 로직)
- **기준점수**:
  - 3세 → 90점 이상
  - 5세 → 85점 이상
  - 7세 → 80점 이상
  - 10세 → 70점 이상
- **특징**: 프로필 나이에 따라 자동으로 필터링 강도 조정

### 3-3. 자녀 프로필 시스템
- **사용 기술**: DiceBear API, localStorage, JSON 파일 DB
- **기능**: 최대 4개 프로필 생성, 성별별 아바타 자동 생성
- **저장 데이터**: 이름, 나이, 성별, 아바타 시드, 시청 시간 제한

### 3-4. 시청 기록 저장 및 필터링
- **사용 기술**: Express REST API, history.json
- **기능**: 영상 시청 시 자동 저장, 프로필별 필터링
- **저장 데이터**: videoId, title, channelTitle, thumbnail, totalScore, violence, language, sexual, educational, summary, profileId, watchedAt

### 3-5. 배지 시스템
- **사용 기술**: Express REST API, badges.json
- **배지 종류**: 10종 (첫 발걸음, 새싹 탐험가, 시청 대장, 안전 보안관 등)
- **기능**: 조건 달성 시 자동 배지 부여 + 획득 팝업 표시

### 3-6. 부모 대시보드
- **사용 기술**: Recharts, React state
- **기능**:
  - 요약 카드 (총 시청 영상, 평균 안전도, 오늘 시청 시간)
  - 안전도 분포 막대 차트
  - 프로필별 시청 기록 탭 필터링
  - 시청 시간 제한 설정

### 3-7. AI 요약 모달 (VideoModal)
- **사용 기술**: React 컴포넌트, props 패턴
- **기능**: 영상 카드 클릭 시 항목별 점수 + AI 요약 모달 표시
- **UI**: 점수 바 (색상으로 안전도 시각화), 말풍선 스타일 요약

### 3-8. 반응형 디자인
- **사용 기술**: Tailwind CSS 반응형 클래스
- **적용**: 모바일 1열 → 태블릿 2열 → PC 3열 그리드
- **테스트**: 실제 모바일 기기에서 확인 완료

### 3-9. 공통 NavBar 컴포넌트
- **사용 기술**: React 컴포넌트, React Router useNavigate
- **기능**: 뒤로가기 버튼, 페이지 타이틀, KidSafe 로고
- **특징**: 모바일/PC 반응형, sticky top 고정

## 5. 면접 예상 질문 & 답변

**Q. AI를 어떻게 활용했나요?**
> Anthropic API를 통해 영상 제목과 설명을 입력하면 폭력성, 언어, 선정성, 교육성 4가지 항목을 분석해 0~100점으로 점수화합니다. 단순 키워드 매칭이 아닌 문맥을 이해하는 LLM의 특성을 활용했습니다.

**Q. YouTube Kids와 뭐가 다른가요?**
> YouTube Kids는 사전 검수된 라이브러리를 제공하고 왜 안전한지 설명하지 않습니다. KidSafe는 LLM이 실시간으로 영상의 맥락을 분석하고 항목별 점수와 이유를 부모에게 시각적으로 제공합니다.

**Q. 반응형은 어떻게 구현했나요?**
> Tailwind CSS의 반응형 클래스(md:, lg:)를 활용해 모바일 1열, 태블릿 2열, PC 3열 그리드를 구현했습니다. 실제 모바일 기기에서 직접 테스트하며 UI를 개선했습니다.
