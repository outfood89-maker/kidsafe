## 기획서 1 — 분석화면 아이 뷰 간소화

**목적:** 7축 분석 그래프는 부모·심사위원용 증거이지 아이의 UX가 아니다. 아이에게는 "봐도 되는지"만 크고 단순하게.

**현재:** 아이 화면(VideoModal)에서 영상 클릭 시 7축 점수·사유가 그대로 노출.

**변경:**
- 아이 기본 화면 = 큰 신호등 + 키디 한 줄: **"키디가 미리 봤어! 안심하고 봐도 돼 🦕"** (차단 시: **"이건 키디가 아직 안심 못 했어. 다른 거 보러 가자!"**) — 카피 팀장 확정본, verbatim
- "자세히 보기" 탭 → 기존 7축 분석이 그대로 펼쳐짐 (기존 컴포넌트 재사용, 접힘 상태만 추가)
- 부모 진입 경로(있다면)는 기본 펼침 유지

**지키는 것:** 재생 게이팅 로직 불변. 분석 데이터·API 무변경 — 순수 표시 계층.

**공수/위험:** 0.5~1일 / 핵심 모달 수정이므로 회귀 확인 필수 (재생·게이팅·연속재생 스모크).

**심사 논리:** 아동 눈높이 설계의 증거(UX 25점) + 분석력 과시는 '자세히 보기'로 보존.

## 기획서 2 — 키디의 방: 말하기 연습 v0

**목적:** 채팅 로그 UI는 어른의 은유다. 아이는 지난 대화를 안 본다 — 현재형 음성 대화로 재구성해 '말하기 연습' 기능으로 만든다. (오너 발제)

**재사용 자산 (기술 난이도가 낮은 이유 — 오너 판단 타당):** `/chat` 엔드포인트(수준별 가이드·안전 규칙 포함), `useKiddySpeech`(STT), `useKiddyVoice`(TTS), 키디 캐릭터 에셋, P 위기 스크리닝(서버 경유라 자동 적용).

**v0 스코프 (조립 수준, 1~1.5일):**
- 새 라우트 `/kiddy-room` (기존 동선 무수정 — 추가형이라 회귀 위험 낮음)
- 화면: 키디 캐릭터 크게 + "탭하고 말해봐" 버튼 하나. 텍스트 로그 없음
- 흐름: 입장 → 키디 인사(TTS) → 아이가 탭+말함(STT) → `/chat` 응답 → TTS로 대답 → 반복
- **대화 기록 미저장** (세션 메모리만) — 아이도 안 보고, 우리 미저장 철학과 일치

**가드레일 (양보 불가 — 애착 리스크의 정중앙이므로):**
1. 프레임 = **말하기 연습** (친구랑 노는 방 ❌, 언어 연습 ⭕) — 랜딩·덱에서도 이 프레임으로만 소개
2. 세션 리추얼: 8턴 또는 5분 후 키디가 다정하게 마무리 — **"오늘 이야기 진짜 재밌었어! 이제 가서 놀자. 내일 또 이야기하자!"** (카피 확정본). "보고 싶어/외로워/가지 마" 류 애착 유발·죄책감 카피 **절대 금지**
3. 위기 스크리닝 동일 적용 (서버 경로라 P 구현이 자동 커버 — 검증 케이스로 확인만)
4. 리텐션 장치 없음: 방문 배지·연속 방문 보상 금지

**v0 제외 (v1/고도화):** 배경 씬 아트, 대화 주제 카드, 발음·표현 피드백, 수준 선택 UI(백엔드 기본값 사용).

**리스크:** 음성 루프 마이크 상태 처리 — 체크인·챗봇에서 검증된 훅 재사용으로 완화. 미지원 브라우저 폴백(버튼 안내) 필요.

## 기획서 3 — 스스로 일기 쓰기 (고도화 확정, 프리즈 전 착수 불가)

**왜 지금 안 하나:** 난이도가 아니라 **선행 설계 결정 3개** 때문이다. ① 저장·프라이버시 모델(기본 비공개? 아이 열람 UX? 부모 공유 흐름? '비밀 미저장' 철학과의 정합) ② 새 데이터 모델(테이블·보존 기간) ③ 리포트·패턴 감지와 연결할지. 이걸 급하게 정하면 우리가 쌓은 프라이버시 서사가 꼬인다. 고도화 1개월차, 오너+팀장 설계 회의 선행.

**v1 방향 스케치 (덱·신청서용):**
- 체크인 확장형 진입: 체크인 끝에 "오늘을 일기로 남겨볼까?"
- 작성: 음성으로 말하거나, 키디가 선택지를 주며 함께 구성 ("오늘 제일 재밌었던 건? 누구랑 있었어?")
- 완성: 키디가 아이의 말을 일기 문장으로 읽어줌 → 아이가 **"간직할래 / 안 할래"** 선택
- 프라이버시: **기본 비공개 (자물쇠 은유)** — 부모 공유는 아이가 페이지 단위로 선택. "말은 아이의 것" 원칙의 완성형
- 저연령 설계 (오너 우려 반영): 음성·그림 우선, 큰 터치 타겟, 실수 복구 쉬움(되돌리기), 오터치 방지

**서사 가치:** 감정의 언어를 '기록'으로 확장 — 감정 사전과 연동해 아이가 자기 마음의 작가가 되는 경험. 플랫폼 비전("첫 영상 친구 → 아이의 첫 디지털 생활의 홈베이스")의 첫 번째 증거 기능.
