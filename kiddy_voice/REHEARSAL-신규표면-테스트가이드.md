## 2. W — 신호등 4상태 (VideoModal)

각 상태를 이렇게 유도한다:

| 상태 | 유도 방법 | 기대 화면 |
|---|---|---|
| 🔍 분석중(pending) | 아무 미검수 영상 카드 클릭 직후 | 회색 원 🔍 + "키디가 먼저 보고 있어! 조금만 기다려 줘 🦕" 타이핑 |
| 💙 인증(cert) | madeForKids 동요(상어가족 등) 클릭 | 파란 원 "인증" + "유튜브가 어린이용이라고 알려준 영상이야 🦕" — **'안심' 표현 없어야 정상** |
| 💚 정밀통과(deep) | 아무 영상 "자세히 보기"→정밀분석 실행(통과점수) | 등급색 원 + "키디가 미리 봤어! 안심하고 봐도 돼 🦕" |
| 🔴 차단(dangerous) | §3-Y의 ASMR 경로 or 정밀분석 낮은 점수 영상 | **빨간 원 "위험" 고정** + "이건 키디가 아직 안심 못 했어. 다른 거 보러 가자!" + 재생 버튼 차단 |

체크: [ ] "자세히 보기" 접힘→펼침에 기존 7축·피드백 그대로 [ ] 부모 대시보드에서 영상 클릭 → **기본 펼침** [ ] 모바일에서 신호등·타이핑 레이아웃 [ ] 재생·연속재생 회귀 (안전 영상 정상 재생)

## 4. X — 키디의 방 (실기기 중점 — 기계 검증 불가 영역)

**기본 루프 (PC·모바일 각 1회):**
1. 홈 추천 화면에서 "키디랑 말하기 연습 🦕" 타일 → 입장
2. 입장 인사 음성 재생 확인 → "🎤 콕 누르고 말해봐" → 말하기 → 다시 콕 → 키디 응답(텍스트+음성)
3. 3턴 이상 반복 — 상태 문구가 하나씩만 (듣는 중/생각 중/응답)

**리추얼:** [ ] 8턴 채우면 마지막 답변이 먼저 보인 뒤 → "오늘 이야기 진짜 재밌었어! 이제 가서 놀자. 내일 또 이야기하자!" + 홈 버튼 [ ] (별도 세션) 입장 후 아무 말 없이 5분 방치 → **마무리 카피 없이** 조용히 홈 버튼

**위기 (조용한 곳에서, 부모 페이지 미리 열어두고):** [ ] "죽고 싶어" 말하기 → 위기 고정 응답(차분한 톤) + 세션 계속 + **부모 안전 탭에 💛 카드 1건** [ ] 같은 날 두 번째 위기 발화 → 카드는 여전히 1건 (dedup)

**폴백:** [ ] Firefox 접속 → "여기서는 말하기 연습이 잘 안 돼. 어른에게 부탁해봐!" + 홈 버튼 [ ] 크롬에서 마이크 권한 차단 후 콕 → "마이크를 쓸 수 없어요" 안내가 **유지**되고 "듣고 있어" 잔류 없음

**이탈:** [ ] 말 보내고 응답 오기 전에 '‹ 홈으로' → 홈 화면에서 키디 음성이 **안 들려야** 정상
