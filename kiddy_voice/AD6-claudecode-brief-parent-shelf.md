## §0. 전제

1. `feature/diary-v0`. DIARY_V0 게이트·삭제 금지·try-catch·금지 4종 불변. **푸시 금지 유지, 알림은 인앱만(푸시 절대 금지 — 대본 §1 원칙 메모 그대로).**
2. 채널 원칙: 편지는 **부모→아이** 방향(비밀 채널 침식 없음). 도장·편지는 entry에 저장 — **찢기 시 함께 소멸**(아이의 삭제권이 부모 반응보다 우선, 기존 불변식).
3. 도장·편지에 배지·보상·평가 연결 금지. 도장은 이모지 4종만(문구 없는 반응).

## §1. 저장 (diaryStore 확장)

- entry 선택 필드 `stamp: { emoji, letter, at, seenAt }` — `setStamp(pid, entryId, { emoji, letter })`(변경 = 덮어쓰기, `at` 갱신·`seenAt` 리셋), `markStampSeen(pid, entryId)`.
- `letter`는 저장 전 `String(...).slice(0, 30)` 서버급 방어(UI maxLength와 별개).
- 미확인 도장 조회 헬퍼: `getUnseenStamps(pid)` → `[{ entryId, hasLetter }]`.

## §2. 부모 대시보드 — '가족 책장' 열람 (읽기 전용 + 도장·편지만 쓰기)

**파일:** `client/src/pages/ParentDashboard.jsx`
- 좌측 사이드바 탭 배열(≈100~106행)에 `{ id: "shelf", icon: "📖", label: "가족 책장", short: "책장" }` 추가 + `mainTab === "shelf"` 섹션 신설. **DIARY_V0 게이트 뒤에만 탭 노출.**
- 아이별 탭(kiddyTab 패턴 — '전체' 없음, scopedId 잠금 존중). 데이터 = `diaryStore.getEntries(profileId)` **직접 읽기(같은 브라우저 localStorage — v0 한계로 명시 주석: 고도화 Supabase 이전 시 API로 교체)**.
- 렌더 = 월별 2열 그리드 크림 카드 → 월 열람 → 페이지 상세, **FamilyShelf와 동일 문법**(그리드·크림카드·날짜|기분 라인·밑줄 문장·이미지). 단 **읽기 전용**: 찢기·쓰기 버튼 없음. 빈 상태 문구는 부모 톤 안내 1줄(§5 스탬프).
- **도장 UI(페이지 상세)**: 이모지 4종 `❤️ 👍 🌟 🐾` 선택(페이지당 1개, 재선택 = 변경). 선택 시 크림 카드 우하단에 목업 ③ 도장 문법(점선 원형·기울임)으로 표시.
- **편지 입력(도장과 함께, 선택)**: 1줄 입력, `maxLength=30` + 카운터(`n/30`). placeholder = `아이에게 전할 한마디 (30자)` / 아래 서브 넛지 = `칭찬보다, 마음을 전해보세요 — "네 이야기 잘 읽었어" 처럼요.` (둘 다 팀장 verbatim). 저장 버튼 1개(도장+편지 함께 setStamp).

## §3. 아이 쪽 — 도장·편지 표시 + 키디 낭독

**파일:** `client/src/pages/FamilyShelf.jsx`
- 페이지 상세: `entry.stamp` 있으면 크림 카드 우하단에 도장(이모지, 목업 ③ 문법). **편지 본문은 기본 비표시** — 편지 있으면 도장 옆 `✉️` 표시.
- `✉️` 탭 → 키디 낭독: `LETTER_READ` 카피 표시+TTS(`voice.speak`) → 이어서 **편지 본문 낭독 + 화면 표시**(부모 작성 텍스트 — 카피 게이트 대상 아님). 마운트 시 `markStampSeen` 처리.
- 월 목록 행에도 도장 이모지 노출(있으면) — 시각만, 문구 없음.

## §4. 아이 알림 (다음 자연 접속 인앱 1회)

**파일:** `client/src/pages/KidHome.jsx` (⚠️ 커밋 시 해인 절차)
- 진입 시 `getUnseenStamps(pid)` 확인 → 있으면 **인앱 카드 1개**(티저 말풍선과 같은 자리 문법, 동시 표시 금지 — 알림이 티저보다 우선): 도장만 = `STAMP_NOTICE.stamp` / 편지 포함 = `STAMP_NOTICE.letter` (§5). 탭 → `/family-shelf` 이동(해당 페이지 열람 시 seen 처리로 자연 소멸). 닫기(✕)도 허용 — 닫아도 다음 접속에 다시 뜨는 건 미확인이 남아 있는 동안 1회/접속.
