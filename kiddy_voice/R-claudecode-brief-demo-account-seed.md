# [Claude Code 작업 지시서] R — P0 2차: 심사위원 데모 계정 + 시드 데이터

> **왜:** 제출물 = **URL + 서면**. 심사위원이 직접 접속해 만져보는 게 심사의 전부 — 텅 빈 계정에서 시작하면 기술 완성도 25점이 직격. **접속 즉시 체크인→리포트→감정 그래프가 살아있어야** 한다. (팀장 메모 2호 조정 1)
> **이중 용도:** 같은 계정·시드로 **7/12 시연 영상 녹화** ([demo-video-scenario-v1.md](demo-video-scenario-v1.md) 전제 4·5).
> **일정:** 계정+기본 시드 = 2차 웨이브(~7/8) / 최종본(배지·패턴 반영 확인) = 7/11.
> **막히면 임의로 우회하지 말고 멈추고 보고.**

## 0. 설계 결정 (컨트롤 타워)

- **전용 신규 계정 1개** (기존 Freddie 계정 재사용 금지 — 테스트 잔재 분리). 앱 회원가입으로 생성 — **확정 (2026-07-02 Freddie): `demokiddy@naver.com` / `kiddy123`** (신청서에 기재할 값. 이메일 인증 불요 확인됨).
  - ⚠️ 신청서 제출 전까지 이 계정 정보는 외부 비공개 (시드 오염 방지).
- **아이 프로필 2개** (예: "하늘" 6세 / "바다" 5세). ⚠️ **이유: 체크인은 프로필당 하루 1번** — 심사위원 여러 명이 같은 날 접속하면 두 번째 사람은 체크인을 못 봄. 프로필 2개 + 안내 문구(§4)로 방어.
- **시딩 방식 하이브리드:**
  - **과거 체크인(3주치)** = SQL 삽입 (앱으로는 과거 날짜 생성 불가) — 작업자가 SQL 파일 작성, **Freddie가 대시보드 실행**.
  - **시청 기록·검색·찜·미니게임** = 앱 실제 조작 (Freddie 또는 작업자 안내에 따라) — 검수 캐시(analysis_cache)가 자연스럽게 채워져 시청 분석·코치가 진짜 데이터로 동작. SQL로 history만 넣으면 캐시 조인이 비어 반쪽이 됨.
  - **배지** = 직접 삽입 금지 — 시드 완료 후 앱 진입·트리거로 **자연 획득** (새 체계 검증을 겸함).

## 1. 시드 체크인 설계 — "하늘" 프로필 (3주치, 스토리 있는 데이터)

시연 영상·패턴 감지·대화의 씨앗·hadSecrets가 **전부 이 데이터에서 살아나야** 한다. 날짜는 실행일 기준 상대값(D-21 ~ D-1)으로 SQL 작성.

| 구간 | 패턴 | 목적 |
|---|---|---|
| D-21 ~ D-15 (1주차) | happy 4 · good 2 · soso 1 | 평온한 기준선 |
| D-14 ~ D-8 (2주차) | good 3 · soso 2 · **sad 2** | 부정 감정 비율 상승 시작 |
| D-7 ~ D-1 (3주차) | good 2 · soso 1 · **sad 3 · angry 1** | 2주 연속 상승 → **패턴 감지 신호 발화** + "슬픔 섞인 주간"(영상 장면 4) |

- **공유(share=true) 하이라이트**: 최근 주에 2건 이상, answers에 아이다운 화제 포함 — 예: `[{qId:"day", answer:"공룡 그림 그렸어"}, {qId:"watch_genre", answer:"공룡"}]` → **대화의 씨앗이 "공룡" 화제로 발화**(영상 장면 4-4).
- **비공유(share=false)**: 3주차에 1~2건 → hadSecrets("비밀도 있었어요 🤫") 발화. 🚨 **O 정책 준수: share=false 행은 `answers`를 반드시 `'[]'`로** 삽입 (마스킹 정책과 모순되는 시드 금지).
- followup 넣을 경우 secret=true면 `{q, secret:true}`만 (O 조건 1 동일).
- mood_emoji는 MOOD 코드와 일치(😄🙂😐😢😡). user_id·profile_id는 신규 계정 것.
- "바다" 프로필은 가볍게 1주치(happy 위주)만 — 프로필 전환 시 비어 보이지 않을 정도.

## 2. 앱 조작 시드 (SQL 아님 — 체크리스트로 안내)

작업자는 **조작 체크리스트**를 산출물로 작성 (Freddie가 따라 하거나 직접 수행):
1. "하늘"로 검색 3~4회 (공룡·동요·과학 등 — 검색 배지·장르 개척자 재료)
2. 안전 영상 5~8개 재생 (검수 신호등 초록 위주 — 시청 분석·코치·"이번 주 본 것" 재료)
3. 찜 2~3개, 미니게임 2~3판 (game_bonus — 컬렉션에 진행감)
4. 마지막에 배지 체크 트리거 자연 발동 확인 → **컬렉션에 새 체계 배지만** 뜨는지 검증 (제거된 배지 안 보여야)
