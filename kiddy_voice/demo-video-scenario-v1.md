# 시연 영상 시나리오 스크립트 v1 (7/12 녹화용 — 클릭 단위)

*작성: 컨트롤 타워 · 2026-07-02 (팀장 메모 4호 §4 기준, 즉시 액션 ①)*
*총 목표 길이: 3~5분. 내레이션 원고·편집은 오너·팀장 협의(이 문서는 화면·클릭·타이밍만).*
*구조 원칙: 기능 나열이 아니라 **하루의 흐름** — 아이의 낮 → 부모의 저녁.*

## 0. 녹화 전제 (7/12 전까지 완성돼야 하는 것 — 없으면 그 장면 촬영 불가)

| # | 전제 | 담당 브리프 | 상태(작성 시점) |
|---|---|---|---|
| 1 | 비밀 약속 멘트 | M | ✅ 구현됨 |
| 2 | 랜딩 새 문패(제목·메타·히어로 카피) | 3차(랜딩 수리) | ⬜ |
| 3 | 대화의 씨앗 카드 | L | ⬜ |
| 4 | 시드 데이터(감정 타임라인 2~4주치, "비밀 있던 주") | 3차(시드) | ⬜ |
| 5 | 데모 계정(체크인 안 한 오늘 상태로 준비) | R(예정) | ⬜ |
| 6 | 시청 종료 '약속' 문구 | ⚠️ P1 보류 ↔ 메모4 시나리오 충돌 — **팀장 확인 대기** | ⬜ |
| 7 | 배지 개편 반영(배지 화면이 잠깐이라도 비치면 개편본이어야) | N | 진행 중 |

- **녹화 계정 주의:** 체크인은 하루 1번 게이트 — 녹화 시작 전 **그날 체크인 안 한 프로필**로. NG나면 같은 날 재촬영 불가 → **예비 프로필 2~3개** 만들어두고 테이크마다 교체.
- **음성(TTS) 포함 여부 미결**(강의실 음향 — 오너 확인 대기). → 아래 지시대로 **음성 켜고 녹화**하되, **모든 키디 대사에 자막**을 입혀 무음 재생도 성립하게 편집. (음향 불확실 시 자막이 보험)
- 녹화 환경: 데스크톱 브라우저 전체화면(주소창 노출 최소화), 알림 끄기, 마우스 커서 강조 옵션 권장. 모바일 프레임 연출이 필요하면 브라우저 반응형 모드(iPhone 프레임)로 통일.

## 장면 1 — 랜딩 → 프로필 선택 (목표 ~30초)

| 클릭/행동 | 화면에 보여야 하는 것 | 메모 |
|---|---|---|
| 1-1. 랜딩 접속 (스크롤 없이 2초 정지) | **새 문패**: 히어로 카피("처음 영상을 만나는 그 순간, 아이가 혼자가 아니도록" 계열) | 브라우저 탭 제목도 새 카피 — 탭이 화면에 보임 |
| 1-2. 천천히 1스크롤 | 공감 섹션 → Meet Kiddy("부모님과 아이를 잇는 다리") | 빠르게 지나가기 — 랜딩은 30초 안에 |
| 1-3. 시작하기 → 로그인(데모 계정) → 프로필 선택 | 아바타 프로필 카드 | 로그인 입력은 편집에서 배속/컷 |

## 장면 2 — 아이 체크인: 데모의 심장 (목표 ~90초) ★하이라이트

| 클릭/행동 | 화면에 보여야 하는 것 | 메모 |
|---|---|---|
| 2-1. 홈 진입 → 체크인 오버레이 자동 등장 | 키디 환영 인사(타이핑+음성) | 어제 기분 끌어오는 인사면 최상(시드 전제) |
| 2-2. 기분 선택 — **😢 슬픔** 선택 | 키디가 **차분하게** 받아줌(폭죽 없음 — 사람이 쓴 문장) | 발표 '윤리와 절제' 챕터와 연결되는 컷 |
| 2-3. 후속 질문(한 박자 더) → 칩 선택 | 부드러운 후속 질문 | 이유 칩 중 평범한 것 선택(비밀 칩은 여기선 X) |
| 2-4. 하루/볼 것 질문 → 🎤 직접 말하기(STT) | 말하기 → 되읽기 확인 | K 구현 전제. 불안정하면 버튼 선택으로 대체 가능(백업 동선) |
| 2-5. 공유 선택 화면 → **"비밀이야 🤫" 클릭** | **키디 비밀 약속 멘트** 타이핑+음성: "알았어, 우리만의 비밀이야!…" | ★영상의 클라이맥스 1 — 여기서 1~2초 여유 있게 |
| 2-6. ⭐ 보상 → "영상 보러 가자" | 보상은 공유 여부 무관(연출 설명은 내레이션 몫) | |

## 장면 3 — 검색 → 검수 → 시청 (목표 ~60초)

| 클릭/행동 | 화면에 보여야 하는 것 | 메모 |
|---|---|---|
| 3-1. 검색(아이 관심사 키워드, 예: 공룡) | 검색 중 키디 로더 | 쿼터 아껴야 하면 캐시된 키워드 사용 |
| 3-2. 결과 목록 | **검수 신호등**(90/70/69 안전 뱃지) | 초록 위주 + 노랑 1개 섞이면 설득력↑ |
| 3-3. 영상 재생 (5초만) | 플레이어 | 저작권 고려 — 영상 화면은 짧게, 편집에서 배속 |
| 3-4. 시청 종료/제한 도달 장면 | ⚠️ '키디와의 약속' 문구 (전제 6 — 팀장 확인 대기) | 미구현 시 이 컷 생략하고 J 팁(재미 팩트+영어)으로 대체 가능 |

## 장면 4 — 부모 리포트: 뭉클의 순간 (목표 ~60초) ★하이라이트

| 클릭/행동 | 화면에 보여야 하는 것 | 메모 |
|---|---|---|
| 4-1. 프로필 전환 → 부모 페이지(PIN) | 넷플릭스식 프라이버시 분리 | PIN 입력은 컷 편집 |
| 4-2. 키디의 한 주(리포트) 오픈 | 키디 한마디(타이핑) → **감정 타임라인**(시드로 채워진 한 주) | 슬픔이 섞인 주간이어야 4-3이 삶 |
| 4-3. 스크롤: 하이라이트 → **"그리고 ○○만의 비밀도 있었어요. 그건 키디가 지켜요 🤫"** | ★클라이맥스 2 — 장면 2-5와 수미상관 | 1~2초 정지 |
| 4-4. **대화의 씨앗 카드** "오늘 저녁, 이렇게 말 걸어보세요" | 공유 화제 기반 질문 한 줄 | L 전제 |

## 장면 5 — 마무리 (목표 ~15초)

- 화면: 랜딩 히어로 또는 리포트 카드에서 페이드아웃.
- 내레이션 자리: **저녁 8시 장면의 반전** — "오늘 어땠어?" 대신 "오늘 이런 기분이었다며?" (카피는 팀장 덱과 통일)

## 컷 우선순위 (길이 초과 시 자르는 순서)

3-3(재생) → 1-2(랜딩 스크롤) → 2-4(STT) → 3-4(약속 문구). **2-5(비밀 약속)와 4-3(비밀 지킴)·4-4(씨앗)는 절대 사수** — 이 영상의 존재 이유.

## 남은 결정 (녹화 전까지)

1. TTS 음성 포함 여부 — 오너의 강의실 음향 확인 (미확인 시: 음성 포함 + 전체 자막)
2. 전제 6(시청 종료 '약속' 문구) — 팀장 확인 (3차 편입 vs 컷 대체)
3. 시드 데이터의 '슬픔 섞인 주간' 구체 패턴 — R/시드 브리프에서 확정 (패턴 감지 신호가 리포트에 뜨는 시나리오면 더 좋음)
