# KidSafe 비(非)유튜브 아동 콘텐츠 소스 조사 리포트

> 딥 리서치(6각도 팬아웃 · 소스 26개 fetch · 주장 126개 추출 · 25개 3중 적대검증 → 23 confirmed / 2 refuted) 결과.
> 2026-07-01 조사. 컨트롤 타워 세션.

## 1. 핵심 요약

KidSafe가 유튜브 외 아동 콘텐츠를 도입할 때, **지금 당장 1인 개발자가 라이선스 비용·계약 없이 붙일 수 있는 현실적 선택지의 핵심은 "오픈 라이선스(CC-BY 4.0) 디지털 그림책"과 "퍼블릭 도메인 오디오/텍스트"** 두 축이다. 대표적으로 Pratham Books의 **StoryWeaver**(CC-BY 4.0, 6만4천여 권·370개 이상 언어, 한국어 포함)와 **Book Dash**(CC-BY 4.0 그림책), 텍스트의 **Project Gutenberg**(7만5천여 권 퍼블릭 도메인), 오디오의 **LibriVox**(퍼블릭 도메인 오디오북, 정식 카탈로그 API 제공)가 계약·수수료 없이 재사용 가능하다. 국내에서는 정부 공공데이터포털이 **국립어린이청소년도서관 사서추천도서** OpenAPI를 무료로 제공하지만, 이는 콘텐츠 본문이 아니라 "추천 서지 메타데이터"라는 점에 주의. 반대로 **유튜브를 유지하는 경우**, 자녀 대상(Child-Directed) 앱은 임베드 영상마다 Made-for-Kids 상태를 API로 조회해 추적을 끄고 COPPA·GDPR을 준수해야 하며, 광고 제거·플레이어 개조·재생 게이팅이 금지되어 "안전하게 개조된 임베드"를 만드는 데 구조적 한계가 있다. 결론적으로 KidSafe의 정체성("아이의 첫 미디어 경험·부모 신뢰·키디와의 관계")에는 **광고 없고 라이선스가 명확한 오픈 그림책/오디오를 유튜브 옆에 얹는 "보완(+α)" 전략**이 가장 부합한다.

## 2. 축 1 — 유튜브 대체 vs 보완

### 2-1. 유튜브를 "완전 대체"하기 어려운 이유
유튜브의 강점은 **콘텐츠 양(무한)·검색·무료**. 단일 소스로 완전 대체하려면 방대한 카탈로그·스트리밍 인프라·라이선스 정리를 모두 자체 확보해야 해 1인 개발자엔 비현실적. 다만 콘텐츠 종류를 "영상"에서 "그림책·오디오·동요"로 넓히면 유튜브 없이도 "첫 미디어 경험"을 구성할 오픈 소스가 충분히 존재한다.

### 2-2. 유튜브 유지+보완(+α) — KidSafe에 가장 현실적
유튜브를 영상 축으로 유지하되 그림책·오디오·동요를 별도 소스로 애그리게이션. 이유: (1) 안전성 — 오픈 소스는 광고 없고 콘텐츠 고정이라 유튜브 임베드의 광고·알고리즘 리스크 우회(유튜브는 API 클라이언트의 광고 제거·차단을 금지). (2) 차별화 — 키디가 그림책/오디오를 함께 읽어주는 경험이 정체성(B)에 직접 기여. (3) 비용/난이도 — 오픈 라이선스는 계약·수수료 없어 진입장벽 낮음.

### 2-3. 접근별 트레이드오프
| 접근 | 콘텐츠 양 | 안전성 | 비용 | 기술 난이도 | 차별화 |
|---|---|---|---|---|---|
| 유튜브 완전 대체(자체 카탈로그) | 낮음 | 높음 | 높음 | 높음 | 높음 |
| 유튜브 유지 + 오픈 소스 보완 | 높음 | 중~높음 | 낮음 | 중간 | 높음 |
| 유튜브만 유지 | 매우 높음 | 중간(광고·알고리즘 제약) | 낮음 | 낮음 | 낮음 |

## 3. 축 2 — 콘텐츠 종류별 실제 소스

### (a) 동화·그림책
- **StoryWeaver (Pratham Books, 인도) — 오픈 라이선스, 최우선 추천.** 5.3만~6.4만 권/300~370개 언어, 전부 **CC-BY 4.0**(일부 CC-BY-SA/CC0). 읽기·다운로드·번역·각색·재배포·**상업 이용 허용**, 의무는 출처 표기뿐. **한국어 지원**(전래동화 "흥부와 놀부" 등 100권+, 단 보조적 수준). 광고 없음, 디지털 공공재. ⚠️ 스토리별 라이선스 확인 / "Read Along" 영상은 CC-BY-NC-ND(상업 불가). (storyweaver.org.in/about)
- **Book Dash (남아공) — CC-BY 4.0** 그림책 200+종, 원본 파일 제공, 번역·판매까지 허용. (bookdash.org/the-power-of-open)
- **국립어린이청소년도서관 사서추천도서 (문체부, 한국) — data.go.kr 무료 OpenAPI.** ⚠️ **메타데이터(서지)이지 본문 아님.** 큐레이션 시드로만. (data.go.kr/data/15104976)
- **Open Library (Internet Archive) — 무료 공개 API(무키).** ⚠️ 서지·표지 메타데이터, 본문 아님. 레이트리밋 있음. (openlibrary.org/developers/api)

### (b) 학습·교육(에듀테크)/OER
- StoryWeaver 자체가 OER(다국어 리터러시 자원). Project Gutenberg 텍스트도 학습 자원. 일반 OER 플랫폼(OER Commons/MERLOT)은 이번 미검증 → 별도 실사.

### (c) 음악·동요
- **확정된 오픈 스트리밍/라이브러리 없음.** 동요/키즈 음악은 오픈 소스 빈약 → 국내 동요는 라이선싱/제휴 또는 자체 제작 필요 가능성.

### (d) 오디오
- **LibriVox — 퍼블릭 도메인 오디오북 + 정식 API, 즉시 사용 가능.** 모든 녹음 퍼블릭 도메인(판매·방송·리믹스·상업 자유, 크레딧 의무 없음). API: `librivox.org/api/feed/audiobooks/`, XML 기본 + JSON 지원, 무인증. ⚠️ 아동 적합작 개별 선별 필요 / 퍼블릭 도메인은 **미국 기준** → 국내 저작권 재확인. (librivox.org/api/info)
- **Yoto — 스크린프리 오디오(하드웨어+대형 브랜드 라이선스).** Disney·Peppa·Harry Potter 등 1,000+종. 직접 붙일 소스 아님 → **UX/포지셔닝 벤치마크**("광고·카메라·마이크 없는 안전 오디오+부모 신뢰").

### (텍스트) Project Gutenberg
- 7.5만+ 퍼블릭 도메인 이북, 상업 재사용 자유(상표 미사용 시). ⚠️ "아동 고전이 인기 목록에 직접 노출" 세부 주장은 **검증에서 반박** / 미국 기준 / 텍스트라 삽화·오디오화 가공 필요.

## 6. 축 3 — 라이선스·저작권·규제
### 6-1. 오픈 라이선스 사례
| 유형 | 소스 | 조건 | 상업 |
|---|---|---|---|
| CC-BY 4.0 | StoryWeaver, Book Dash | 출처 표기 | 허용 |
| 퍼블릭 도메인(미국) | Gutenberg, LibriVox | 무조건(상표 제외) | 허용 |
| 공공데이터(국내) | 국립도서관 OpenAPI | 약관 확인 | 재확인 |

### 6-3. 유튜브 유지 시 하드 의무 (Google 공식 개발자 정책)
1. **Made-for-Kids 상태 조회 의무**(`videos.list`의 `status.madeForKids`) + 추적 오프 + COPPA·GDPR 준수.
2. **Child-Directed 통지 + 광고 제약**(개인화·리마케팅 타게팅 금지).
3. **쓰기 액션 전면 금지**(업로드·댓글·재생목록 등, 유도도 불가).
4. **플레이어·광고 개조 및 게이팅 금지** → **광고 제거·플레이어 교체·시청 전 강제 절차 삽입 불가.**

→ **시사점:** 유튜브 임베드로는 "광고 없는 순수 안전 경험"을 만들 수 없다. "안전·부모 신뢰" 정체성 극대화하려면 **비유튜브 오픈 소스(광고 없음) 병행 필요.**

## 7. KidSafe 정체성 시사점·추천
1. **StoryWeaver(그림책) = 최우선.** 광고 없음+CC-BY+한국어+대규모 → "키디가 함께 읽어주는 첫 그림책 경험"에 최적. 정체성(키디 묻고→아이 답하고→부모 뭉클)에 직접 부합. ⚠️ 스토리별 라이선스 확인+출처 표기 UI.
2. **Book Dash = 보조 그림책.**
3. **LibriVox = "자기 전 오디오" 축.** API 쉬움, 무료. 단 아동 적합작 선별+국내 저작권 확인.
4. **국립도서관/Open Library 메타데이터 = 큐레이션 시드**(사서 추천 → 부모 신뢰). 본문 아님을 명확히.
5. **유튜브 = 영상 축 유지하되 규제 의무 이행**, 광고 없는 경험은 오픈 소스로 보완.
6. **Yoto = UX/포지셔닝 벤치마크**(직접 소스 아님).

**추천 로드맵:** 1단계(지금) StoryWeaver 한국어 그림책 큐레이션+키디 낭독/질문 연동 → 2단계 Book Dash+LibriVox 아동 오디오 선별 → 3단계 사서추천/Open Library 메타데이터로 큐레이션 신뢰 강화 → 4단계(제휴 검토) 국내 상용 동요/그림책(티어②), 장기 브랜드 제휴(티어③).

## 8. 불확실·주의 (Caveats)
- **반박된 주장①:** Gutenberg 아동 고전이 "인기 다운로드에 직접 노출"된다는 세부 주장 반박(1-2). 존재는 사실이나 노출 형태 미확정 → 아동 적합작 개별 선별.
- **반박된 주장②:** 국립도서관 OpenAPI "라이선스 범위 제한 없음" 반박(1-2) → **약관/라이선스 범위를 data.go.kr에서 재확인**, 무제한 상업 재사용 단정 금지.
- **퍼블릭 도메인 관할:** LibriVox·Gutenberg PD는 **미국 기준.** 한국(사후 70년 등)에서 보호 대상일 수 있음 → 국내 재사용 전 **국내 저작권 상태 확인**(LibriVox가 사용자 책임으로 명시).
- **StoryWeaver 스토리별 라이선스** 개별 확인 / "Read Along" 영상은 상업 불가.
- **한국어 물량** StoryWeaver 100권+ 수준(보조적) → 주력 한국어는 국내 소스 별도 확보.
- **메타데이터 vs 콘텐츠 혼동 금지**(Open Library·사서추천 = 서지 정보).

## 9. 남은 질문 (추가 실사)

2. 동요/키즈 음악 오픈 소스 — 확정 부재. 라이선싱/자체 제작 판단.
3. **공공누리(KOGL)·국내 공공저작물** 아동 콘텐츠 실물 범위 — 개별 미검증.
4. 일반 OER 플랫폼(OER Commons/MERLOT) 아동 콘텐츠 실물 — 미검증.
5. 국내 저작권법상 미국 퍼블릭 도메인 작품의 한국 내 지위 — 작품별 법률 검토.
6. 국립도서관 OpenAPI 약관 세부(상업 재사용 범위) — 재확인.
