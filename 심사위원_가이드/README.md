# 🦕 Kiddy 심사위원 데모 가이드

> **Kiddy(키디)** — 아이의 첫 미디어 경험을 함께하는 AI 친구.
> 유튜브 영상을 AI가 검수해 안전한 것만 보여주고, 공룡 친구 '키디'가 아이의 하루를 묻고, 그 대답이 그림일기가 되어 부모에게 전해지는 서비스입니다.

---

## 📱 시작하기 전에 — 꼭 읽어주세요

1. **이 서비스는 모바일 웹에 최적화되어 있습니다.**
   **스마트폰 브라우저(사파리·크롬)로 접속하시는 것을 강력히 추천드립니다.** 아이가 실제로 쓰는 환경이 스마트폰이기 때문에, 화면·터치·음성 모두 모바일 기준으로 설계되었습니다. (PC로도 동작은 합니다.)
2. **소리를 켜 주세요.** 🔊 키디가 실제 목소리로 말하고(음성 합성), 배경음악이 흐릅니다 — 이 서비스 경험의 절반은 소리입니다. 무음 스위치를 해제하고 볼륨을 올려주세요.
3. **마이크 권한을 '허용'해 주세요.** 아이가 키디와 말로 대화하는 기능이 있습니다. 권한 요청이 뜨면 허용을 눌러주셔야 음성 기능을 체험하실 수 있습니다.
4. 첫 접속 시 서버가 깨어나느라 **첫 응답이 몇 초 느릴 수 있습니다.** 한 번만 기다려 주시면 이후엔 빠릅니다.

## ⏱ 시간이 3분뿐이시라면

로그인 → 아이 카드의 **🔒 버튼**으로 부모 대시보드 진입(PIN 입력) → **🦕 둘러보기** 버튼

→ 예시 가족의 화면으로 키디의 전체 기능을 **안내와 함께 자동으로** 둘러보실 수 있습니다.

## 🚶 추천 체험 동선 (약 10분)

| 순서 | 어디서 | 무엇을 |
|---|---|---|
| ① | 첫 화면 | 랜딩 페이지에서 서비스 소개 훑기 → 로그인 |
| ② | 프로필 선택 | 아이 프로필(하늘 6세 또는 바다 5세) 선택 → **오늘의 체크인** — 키디와 감정을 나눠보세요. '비밀이야'를 고르면 키디의 약속을 볼 수 있습니다 |
| ③ | 아이 홈 | **영상 검색** — 검색 결과 전체에 자동으로 붙는 **안전 신호등과 점수**를 확인해 보세요 (AI 검수) |
| ④ | 아이 홈 → 📖 | **오늘의 그림일기** — 키디의 질문에 말이나 선택으로 답하면 하루가 일기가 됩니다. **직접 낙서를 그리면 키디가 이어서 완성해 드려요 (강력 추천!)** |
| ⑤ | 가족 책장 | 완성된 일기가 책장에 꽂힌 것 확인 |
| ⑥ | 부모 대시보드(🔒) | **'키디의 한 주' 편지**와 대화의 씨앗 확인 → 방금 만든 그림일기에 **도장과 짧은 편지**(음성 편지도 가능!) 남기기 |
| ⑦ | 아이 화면으로 복귀 | **키디가 부모님의 편지를 아이에게 목소리로 읽어줍니다** — 이 서비스의 심장입니다 💚 |

## 🎁 더 보실 만한 것들

- **키디의 방** — 아이가 키디와 음성으로 자유 대화 (마이크 권한 필요)
- **미니게임 → 📖 단어장** — 그림을 콕 누르면 키디가 단어를 읽어줍니다 (7종 미니게임)
- **배지 컬렉션** — 아이의 활동이 배지로 쌓입니다
- **부모 대시보드의 각 탭 "?" 버튼** — 그 탭의 기능만 짧게 안내해주는 미니 투어
- **시청 분석 탭** — 실데이터 기반 시청 패턴 차트 + AI 코치의 조언

## 🛡 설계 원칙 (심사 참고)

- **검수가 기본기**: 모든 영상은 키워드·자막·썸네일(Vision)·채널 신뢰도까지 다층 검수를 거쳐 90점 이상만 '안전' 표시됩니다.
- **아이의 비밀은 지켜집니다**: 키디와 아이의 대화 원문은 부모에게 전달되지 않습니다. 아이가 **공유를 선택한 것과 감정 흐름 요약만** 전해집니다.
- **사실은 코드가, 분위기만 AI가**: 시간·일정·수치 등 사실 정보는 AI가 지어내지 않도록 설계했습니다.
- 그림일기 데이터는 **기기 로컬 저장**입니다 — 체험하신 일기는 그 기기에만 남습니다.
